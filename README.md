## 目的
用于翻译mvel的源码，进行源码性注释，以方便进行阅读。同时了解其内部的一些实现以及处理过程

## 注意事项
1. 此注释过程将保证不会对原来的代码进行改动，并且相应的注释均是在代码的上方使用//或者是添加到原有的代码当中，使用新增行的方式。
2. 每一版注释将与特定的版本相对应
